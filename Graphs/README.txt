The folders you see in this directory contain the plots for the three architectures:
	GoogLeNet
	ResNet-34
	VGG-16

Each folder contains 12 plots (3 plots per run). The plots are named using the following manner:
	<plot.type>_<learning.rate>_<epochs>.png

where plot.type values include:
	Heatmap
	Accuracy
	Loss

learning.rate values include:
	0001 which indicates 0.001
	001 which indicates 0.01

and epochs values include:
	10
	25
